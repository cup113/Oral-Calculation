口算练习

使用说明

本产品由三大界面组成：

开始界面 (Welcome)

由于设置会自动保存，除初次进入外，前三步执行根据是否需要更改而定。

  1. 选择“类别”。
  2. 对“题数”“避免重复题”“生成题目”等项进行配置。
  3. 配置题型参数。
  4. 点击“提交”，进入练习界面

练习界面 (Exercise)

如果你频繁练习一种题型，可以直接收藏此页面。此页面的网址(URL)已经保存了题型参数和数量等信息。

为了防止多次重复，以下所有Enter均指在键盘上敲击回车键，或在手机/平板屏幕上点击输入法的“开始”“发送”等键。

  1. 点击“开始”按钮或敲击Enter以开始答题。
  2. 根据题目输入答案，并点击“提交”或敲击Enter
  3. 接下来的操作根据是否回答正确而定：
    • 若回答正确，右下角会弹出提示窗口（手机/平板不一定看得到，可观察最上方进度条是否移动，移动说明答对），进入下一步。
    • 若回答错误，正上方会弹出提示窗口，需要立即再次作答。注意，修改时间会计入答题时间。
  4. 一道题答完之后的时间不会计算到答题时间内，可在任意时候再敲击Enter回答下一道题
  5. 等到进度条到达最右侧时，答题结束。正上方会弹出提示窗口，等待跳转到。

报告界面 (Report)

可以截屏保存，也可以点击“返回主页”。

产品特色

  1. 设置自动保存。
  2. 高度可定制化的题型。
  3. 在线计时练习。
  4. 结束后提供全面报告。
